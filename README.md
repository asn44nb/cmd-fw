## API

### Register
```mcfunction
function customcmd:api/register {name:"greet",namespace:"mypack",description:"Greet a player",usage:"greet <player>",permission:1,handler:"mypack:commands/greet"}
```

### Dispatch
```mcfunction
data modify storage customcmd:dispatch command set value "greet"
data modify storage customcmd:dispatch args set value {target:"Notch"}
execute as <player> run function customcmd:api/dispatch
```

### Permission
```mcfunction
execute as <player> run function customcmd:api/set_permission {level:2}
execute as <player> run function customcmd:api/get_permission
```

### List / Unregister
```mcfunction
execute as <player> run function customcmd:api/list
function customcmd:api/unregister {name:"greet"}
```
